淡水美食介紹：吳家阿給蛋餅
在小巷弄裡的吳家阿給蛋餅是位於淡江大學附近的一家受歡迎的早餐店，這家小店以其美味的阿給和粉漿蛋餅聞名。
阿給是一種淡水的特色小吃，是由油豆腐袋中塞入粉條並淋上特製醬料製成。而這家店的蛋餅也非常有特色，外酥內軟，可以選擇多種不同的配料。
粉漿蛋餅是台灣早餐文化中非常受歡迎的一種蛋餅類型，與一般使用麵粉皮製作的傳統蛋餅不同。
粉漿蛋餅的製作方式是將米漿或太白粉漿直接倒在平底鍋上，使其成為薄薄的一層，然後打上雞蛋，並依個人喜好添加各種配料，如蔥花、火腿、起司、培根或蔬菜等。烹調過程中，師傅會將蛋餅摺成半圓形或四方形，使其外層酥脆而內層柔軟。
粉漿蛋餅的特色在於其獨特的口感 - 外層帶有酥脆的質地，而內層則保持著柔軟滑嫩的口感，尤其是蛋液與粉漿結合的部分，吃起來有一種特別的Q彈感。相較於一般麵皮蛋餅，粉漿蛋餅更為輕薄且有嚼勁，風味也更為獨特。
這家店的位置相當便利，就在淡江大學校門附近，步行即可到達，價格相當親民，非常適合學生的預算。
